# Ravine Admin

## Project Introduction
Ravine Admin is a open source project that provides a admin dashboard.

It is built with React.js, Vite, TypeScript, Shadcn UI, and Tailwind CSS.

## Tech Stack
- React.js
- Vite
- TypeScript
- Shadcn UI
- Tailwind CSS
